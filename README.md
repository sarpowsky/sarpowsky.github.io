# My Personal Portfolio Website 🌟  

Hey there! This is my personal portfolio website where I showcase my projects, skills, and a bit about myself. I’ve built it to represent who I am as a developer and to provide an interactive way for people to learn more about me.  

![portfolio](https://github.com/user-attachments/assets/e2fd6596-defd-428f-b488-4e52de863b92)


---

## ✨ Features  
- **Matrix Background**: A subtle, animated matrix effect for a unique touch.  
- **Dark/Light Mode**: Because why not have both?  
- **Interactive Sections**:  
  - _About Me_: A quick intro.  
  - _Projects_: A space to show off what I’ve worked on.  
  - _Skills_: What I bring to the table.  
  - _Devices_: Just for fun—a look at the gadgets I use daily.  
- **Social Links**: Easy access to my GitHub, LinkedIn, and other profiles.  
- **Real-Time Clock**: Because I like having the time right there.

---

## 🛠️ Tech Stack  
This project is built with:  
- **HTML5** for structure.  
- **CSS3** for styling (using some Tailwind for utility classes).  
- **JavaScript** for all the fun stuff like animations and functionality.  

---

## 📂 File Overview  
| File           | What It Does                              |
|----------------|-------------------------------------------|
| `index.html`   | The main HTML file that structures the site. |
| `styles.css`   | Contains all the styling, including responsive design. |
| `scripts.js`   | Manages animations, interactivity, and functionality. |

---

## How to Run It Locally 🖥️  
1. **Clone the repo**:  
   ```bash
   git clone https://github.com/sarpowsky/portfolio-website.git
   cd portfolio-website
2. **Open the project in your browser**:  
   Simply open the `index.html` file in any web browser to view the website.
